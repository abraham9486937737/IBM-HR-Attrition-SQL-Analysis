# Database Setup & Validation

## Objective
To validate the IBM HR Employee Attrition dataset after importing into
Microsoft SQL Server and ensure data quality before analysis.

---

## Database Details
- Database Name: IBM_HR_Emp_Attrition
- Table Name: IBM_HR_Analytics_Employee_Attrition
- Total Records: 1,470

---

## Validation Steps Performed
- Verified table schema and column data types
- Checked total number of employee records
- Ensured EmployeeNumber uniqueness
- Checked for NULL values in key columns
- Validated categorical columns (Attrition, Gender)

---

## Outcome
The dataset is clean and consistent with no duplicate employee records.
It is suitable for further exploratory and analytical SQL queries.
