# Advanced HR Attrition Analysis

## Objective
To apply advanced SQL techniques such as window functions,
ranking, and subqueries to derive deeper HR insights.

---

## Advanced Techniques Used
- Window functions (AVG OVER, RANK)
- Analytical partitions
- Subqueries for data bucketing
- Comparative salary analysis

---

## Key Insights
- Certain job roles show significantly higher attrition rates
- Department and role-based salary averages provide compensation context
- Early-career employees earn less and leave more frequently

---

## Business Value
These insights enable HR leadership to prioritize retention strategies
based on job role, compensation, and employee experience.
